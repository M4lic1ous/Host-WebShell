## 📌 نمای کلی

یک وب‌شل قدرتمند با قابلیت اتصال به ربات تلگرام برای مدیریت از راه دور چندین سرور. شامل مدیریت کامل فایل‌ها و دریافت اطلاعات سرور.

### 🎯 قابلیت‌های اصلی

- **🌐 مدیریت چند سایت** - اضافه و مدیریت چندین وب‌شل
- **📂 مدیریت کامل فایل‌ها**:
  - آپلود/دانلود فایل
  - حذف فایل/پوشه
  - تغییر نام
  - جابجایی
  - کپی کردن
- **📊 دریافت اطلاعات سرور**:
  - اطلاعات سخت‌افزاری
  - وضعیت RAM و CPU
  - اطلاعات دیسک
  - موقعیت جغرافیایی IP
- **📦 دانلود کامل بک‌آپ** به صورت ZIP
- **🔐 احراز هویت** با کلید اختصاصی
- **🤖 کنترل کامل** از طریق ربات تلگرام
